Soy Alberto Cabrera

Un desarrollador Web y Full Stack viviendo y trabajando en Alemania, Bonn.

Me apasiona crear y desarrollar productos limpios, únicos y elegantes. Me especializo en la construcción de sitios web para pequeñas y medianas empresas. Mantengo mi código limpio, legible, modular y bien refactorizado. Me gusta trabajar con interfaces de usuario complejas.

Algunas de la habilidades con las que trabajo en mi día a día:
  • HTML5
  • CSS
  • JavaScript
  • PHP
  • Astro
  • React JS
  • Version Control
  • UI/UX Design
  • Figma
  • Wordpress
