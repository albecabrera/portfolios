Sobre mi

Soy Alberto,

un Fullstack Web Developer y vivo en Alemania, Bonn. Me describo como un desarrollador curioso y apasionado al que le gusta programar plataformas web y, sobre todo, resolver problemas. Mi experiencia en el desarrollo web se basa en la sólida comprensión de HTML para estructurar el contenido, CSS para darle estilo y JavaScript para añadir interactividad dinámica. Mi enfoque es crear sitios web modernos, receptivos y atractivos que brinden una experiencia de usuario excepcional. **Diseño de Interfaz y Experiencia de Usuario:** *Figma y Framer:* Utilizo Figma para crear prototipos de diseño de alta fidelidad, permitiéndome visualizar y perfeccionar la interfaz de usuario antes de pasar al desarrollo. Además, mi habilidad con Framer me permite llevar esos diseños a un nivel superior, agregando interactividad y animaciones para una experiencia de usuario inmersiva. **Diseño Gráfico:** *Adobe Illustrator y Photoshop:* Mi destreza en Adobe Illustrator y Photoshop me permite crear gráficos visualmente impactantes y elementos de diseño para mejorar la estética general de los proyectos. Desde la creación de logotipos hasta la edición de imágenes, estoy comprometido a garantizar que cada detalle contribuya a la coherencia y profesionalismo del diseño. **Modelado 3D y Animación:** *Blender:* Exploro el mundo tridimensional con Blender, creando modelos 3D y animaciones impresionantes. Esta habilidad me permite agregar una dimensión adicional a mis proyectos, proporcionando una experiencia visual única y cautivadora. Al explorar y dominar estas tecnologías, mi objetivo es fusionar la funcionalidad y el diseño, creando soluciones web y gráficas que destaquen y dejen una impresión duradera. ¡Espero que disfrutes explorando mi portafolio tanto como yo disfruto creando estas experiencias digitales!
